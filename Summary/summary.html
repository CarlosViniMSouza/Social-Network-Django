Nesta série de tutoriais de quatro partes, você construirá uma pequena rede social que permite aos usuários postar mensagens curtas baseadas em texto. Os usuários de seu aplicativo podem seguir outros usuários para ver suas postagens ou parar de segui-los para parar de ver suas postagens:

Ao final desta série de tutoriais, você também aprenderá a usar a estrutura CSS Bulma para dar ao seu aplicativo uma aparência amigável e torná-lo uma parte impressionante de seu portfólio de desenvolvimento web que você pode se orgulhar de exibir.

In the first part of this series, you’ll plan the project, create a base Django web app, and extend the built-in user model with a post-save hook.
