Nesta série de tutoriais de quatro partes, você construirá uma pequena rede social que permite aos usuários postar mensagens curtas baseadas em texto. Os usuários de seu aplicativo podem seguir outros usuários para ver suas postagens ou parar de segui-los para parar de ver suas postagens:

Ao final desta série de tutoriais, você também aprenderá a usar a estrutura CSS Bulma para dar ao seu aplicativo uma aparência amigável e torná-lo uma parte impressionante de seu portfólio de desenvolvimento web que você pode se orgulhar de exibir.

In the first part of this series, you’ll plan the project, create a base Django web app, and extend the built-in user model with a post-save hook. At the end of this part, you’ll be able to create new users through the Django admin interface and your app will automatically generate a profile for each new user and set up the necessary connection:

A implementação do usuário de back-end estabelece a base que você construirá nas próximas partes.

Project Overview

Nesta seção, você terá uma ideia sólida do que construirá e por que o fará dessa maneira. Você também investigará as relações de banco de dados que implementará e chegará a um esboço de projeto completo, pronto para ser executado. Em suma, você reservará algum tempo para debater a ideia do seu projeto.

Você pode estar ansioso para começar a programar, mas antes de começar a trabalhar em qualquer projeto de codificação, é útil pensar sobre a estrutura do que você deseja construir.

Você pode usar pseudocódigo, especificações escritas, diagramas de banco de dados, rabiscos de caderno ou qualquer coisa que pareça acessível e o ajude a pensar. Não pule esta parte! É uma etapa essencial para a construção de qualquer projeto. O tempo que você investe em planejamento reduzirá drasticamente o tempo de implementação.

Então, o que você precisa para uma rede social? Em sua forma mais básica, você precisa de duas coisas:

Conexões de usuário para usuário que permitem que as pessoas se conectem umas com as outras
Criação de conteúdo e funcionalidade de exibição para que seus usuários possam criar saídas para que seus usuários conectados vejam

Você pode pensar nesses dois tópicos como separados um do outro, mas precisará de ambos para que sua rede social funcione conforme o esperado.
